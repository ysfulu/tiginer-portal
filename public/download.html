Tiginer'i Kurun

Kurulum paketini indirin, Docker ile dakikalar içinde çalıştırın. Kaynak koda erişim gerektirmez.

Tek Komutla Kurulum

Sunucuda tek satır çalıştırın. Docker yoksa otomatik kurulur, veritabanı başlatılır, admin hesabı oluşturulur.

sudo bash -c "$(curl -fsSL https://tiginer.com/install.sh)"

Ubuntu 22.04+ · Debian 12+ · RHEL 9+ · Docker otomatik kurulur

iex ((iwr 'https://tiginer.com/install.ps1').Content)

Windows Server 2019+ · PowerShell'i Yönetici olarak açın · Docker Desktop önceden kurulu olmalı

Kurulumdan sonra servisleri yönetmek için: tiginer status · tiginer logs · tiginer update

Kurulum Dokümanı İndir

Kurulum Sonrası

  1. Kurulum sırasında girdiğiniz lisans anahtarı portal'da otomatik doğrulanır.
  2. Script size bir admin e-posta ve şifresi gösterir — ekran çıktısını kaydedin.
  3. Tarayıcıdan http://sunucu-ip:3002 adresine gidin ve giriş yapın.
  4. Servisi yönetmek için tiginer komutunu kullanın — Linux'ta bash, Windows'ta PowerShell üzerinden aynı komutlar çalışır (tiginer info, tiginer logs, tiginer stop).
  5. Yeni sürüme geçmek için sudo tiginer update (Linux) veya tiginer update (Windows, yönetici PS) yeterli.
